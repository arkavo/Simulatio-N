Welcome Simulatio-N

This is an N-body simulator using parallel programming for efficiency.

Progress so far

Managed to create a working sequential code which works with great accuracy for upto 10 bodies. Managed to create our own Libraries which would be used in Parallel implementation. Generate a text file and a GNUPlot file for results.

Future Goals

Implement Parallelization. Create an interesting GUI for the Simulator.
